HOUSE SOLUTIONS — جدول العمل السحابي V7

هذه النسخة تجعل جدول العمل مشتركًا بين جميع الأجهزة عبر Supabase، وتضيف:
- علامة إنجاز لكل مهمة، وحالتها متزامنة سحابيًا للجميع.
- موقع المهمة كحقل كتابة يدوي بدل نوع المهمة.
- ظهور موقع المهمة أسفل وصفها داخل الجدول.
- حذف شريط التنقل السفلي الكبير من الجوال.

خطوة مطلوبة بعد التحديث:
1) افتح Supabase Dashboard للمشروع الحالي.
2) افتح SQL Editor > New query.
3) الصق محتوى ملف supabase_schedule_setup.sql الموجود مع المشروع.
4) اضغط Run مرة واحدة.

مهم:
- ملف SQL آمن لإعادة التشغيل.
- لن يحذف المهام القديمة.
- سيضيف فقط عمود task_location وعمود is_completed إن لم يكونا موجودين.
- لا تحتاج لتغيير config.js.
