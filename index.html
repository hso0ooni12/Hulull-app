ربط النظام بقاعدة البيانات السحابية

الواجهة جاهزة. نفّذ هذه الخطوات البسيطة مرة واحدة، وبعدها سيعمل الدخول باسم المستخدم admin.

1
شغّل ملف قاعدة البيانات
افتح Supabase → SQL Editor والصق ملف supabase_setup.sql ثم Run.
2
اربط الموقع
ضع Project URL وPublishable Key داخل config.js.
3
أنشئ حساب الإدارة الوحيد
من Authentication → Users أضف البريد admin@company.local وكلمة المرور Admin1 مع تفعيل Auto Confirm User. داخل الموقع ستدخل باسم admin وكلمة Admin.

مهم: لا تضع Secret key أو service_role في ملفات الموقع.
